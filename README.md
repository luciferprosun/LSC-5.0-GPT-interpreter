# LSC 5.0 GPT Interpreter
This repository contains the LSC 5.0@ — Relativistic Neutrino Detection Framework and associated files.

## Contents
- `LSC_5.0_Framework.md`: Core framework description and formulas.
- Files from the original ZIP archive.

## Simulation

Run the toy energy-reconstruction simulation:

```bash
python3 simulations/simulation.py
```

The script writes CSV and SVG outputs under `simulations/outputs/` and does not require third-party Python packages.
